# Premade Assets

Hedra comes with a number of premade assets, geverally for demos, tests, or quick prototypes. To that end, they all require minimal config. Additionally, all Object assets (like Cube) will use `MeshNormalMaterial` by default, though many will use other materials depending on the props provided.
